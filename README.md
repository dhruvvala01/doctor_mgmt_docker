# 🩺 Doctor Management System (Django + Docker)

A modern, containerized Clinic and Doctor Management System built with Django and PostgreSQL, optimized with a full DevOps pipeline.

---

## 🚀 Features & Tech Stack

* **Backend:** Django (Python 3.12)
* **Database:** PostgreSQL 15 (Containerized)
* **Containerization:** Docker & Docker Compose
* **CI/CD Automation:** GitHub Actions
* **Registry:** Docker Hub (`dhruvvala/doctor_mgmt_web`)

---
